# CoreCleaner – Android App

CoreCleaner is a modern, Play Store–compliant Android app for:
- Junk cleaning
- Duplicate photo detection
- Storage insights
- System monitoring

## Features
- Quick Clean (cache, temp, thumbnails)
- Duplicate Photo Cleaner (Pro)
- Reclaim Space (large files & unused apps)
- System Monitor (RAM, CPU, storage, battery)
- Ads + Premium monetization
- Android 10–15+ compatible

## Tech Stack
- Kotlin
- Jetpack Compose (Material 3)
- MVVM Architecture
- Hilt (DI)
- MediaStore & SAF (future-proof)
- Google Play Billing v6
- AdMob (Banner only)

## Build Instructions
1. Open project in Android Studio
2. Let Gradle sync
3. Build → Generate Signed APK / AAB

## Play Store
This app follows Google Play policies strictly:
- No fake performance claims
- No illegal permissions
- Transparent user control
